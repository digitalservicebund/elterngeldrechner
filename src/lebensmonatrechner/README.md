# Lebensmonatrechner

This "package" provides the logic to calculate the time span of a _Lebensmonat_.
_Lebensmonate_ are a special kind of "months" in relation to the birthday of
a child. It has an important meaning in the various contexts of _Elterngeld_.
While it seem trivial at first, there are some special rules that _Lebensmonate_
follow.
